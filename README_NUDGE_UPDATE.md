# FocusFlow 넛지 시스템 업데이트 내용

## 추가된 주요 기능

1. 집중 상태 단계 분류
   - 1단계: 정상 집중 / 집중 유지
   - 2단계: 집중 흔들림 / 흐름 회복 필요
   - 3단계: 휴식 필요
   - 4단계: 집중 저하 / 강한 복귀 유도

2. 사용자 대시보드 기능
   - 현재 집중 단계 배지 표시
   - 최근 1시간 집중률 표시
   - 상태별 추천 행동 버튼 표시
   - 5분/10분 집중 모드 타이머
   - 1분 호흡/5분 휴식 타이머
   - 추천 솔루션/영상/음악/문구 카드 표시

3. 관리자 기능
   - 넛지 추천 콘텐츠 등록
   - 집중 단계별 콘텐츠 유형 관리
   - 콘텐츠 제목, 설명, URL 수정/삭제
   - 기존 넛지 문구 관리 기능 유지

4. DB 추가 테이블
   - nudge_resources: 단계별 추천 콘텐츠 저장
   - focus_sessions: 집중/휴식 세션 기록 확장용 테이블

## 기존 DB를 이미 만든 경우

기존 DB를 삭제하지 않고 기능만 추가하려면 아래 파일을 MySQL에서 실행하세요.

```sql
SOURCE migration_nudge_update.sql;
```

처음부터 DB를 새로 만들 경우에는 기존처럼 `schema.sql`을 실행하면 됩니다.

## 실행 방법

```bash
npm install
npm start
```

그 다음 브라우저에서 아래 주소로 접속합니다.

- 사용자 화면: http://localhost:3000
- 관리자 화면: http://localhost:3000/admin.html
